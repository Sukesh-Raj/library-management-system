# Library Management System

A console-based **Library Management System** developed using **Core Java, JDBC, and MySQL**.  
This project automates basic library operations such as managing books, registering users, issuing and returning books, tracking due dates, and calculating fines for late returns.

## Features

- Add new books
- View all books
- Update book details
- Remove books
- Search books by title
- Search books by author
- Register library users
- View registered users
- Issue books to users
- Return books
- Track due dates
- Calculate fine for late returns
- View all transactions

## Technologies Used

- **Java**
- **Core Java (OOP concepts)**
- **JDBC**
- **MySQL**
- **Console-based interface**
